API Endpoints

Authentication

  • POST /api/auth/new - Register new user
  • POST /api/auth/recovery - Recover secret key

User Posts

  • POST /api/user/posts/ - Create post
  • GET /api/user/{username}/posts/ - Get user posts
  • GET /api/user/posts/{post_id}/ - Get single post
  • PATCH /api/user/posts/{post_id}/ - Update post
  • DELETE /api/user/posts/{post_id}/ - Delete post

Media

  • POST /api/media/upload - Upload media
  • GET /api/media/list - List user media
  • GET /api/media/{media_id} - Get media info
  • DELETE /api/media/{media_id} - Delete media
